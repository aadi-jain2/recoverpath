# RecoverPath

Landing page for RecoverPath — a chest strap for post-discharge heart failure and COPD monitoring.

## Deploy on Vercel

1. Import this repository in [Vercel](https://vercel.com/new).
2. Leave **Framework Preset** as **Other** (static site, no build step).
3. Deploy — Vercel serves `index.html` from the repo root.

## Local preview

Open `index.html` in a browser, or run:

```bash
npx http-server . -p 8080
```

## Files

- `index.html` — single-page site (HTML, CSS, JS)
- `RC_Logo_TBG.png` — brand logo and favicon
